# Digital Products / Gumroad Research — 2026-04-19

## Research Scope
Re-run of Gumroad digital products opportunity research. Focus:
1. Where the opportunity is
2. What could be automated
3. Where to earn money on western platforms
4. What a code system can build/automate

## Key Findings

### Top Revenue Categories
1. **AI Workflow Templates** (Make.com/n8n) — $29–$99, very low competition
2. **Hyper-Niche Notion/Spreadsheet Templates** — $29–$99, saturated generic, wide open niche
3. **Micro-Courses** — $29–$99, outcome-driven
4. **Developer Resources** — Chrome extensions, code templates, API kits (90%+ growth, 76%+ margins)
5. **Lead Gen & Funnel Kits** — constant demand, few quality products
6. **HR/Compliance Tools for SMBs** — $99–$499/mo subscription

### Price Points That Work
- $9–$19: Impulse buys (presets, single templates)
- $29–$49: Mainstream digital product
- $49–$99: Premium templates, workflow packs
- $99–$297+: Cohort courses, done-for-you systems

### Platform Recommendation
**For Python/dev creators: LemonSqueezy** (developer-friendly, API, license keys, global tax MoR) or **Payhip** (0% fees on Pro plan, instant payouts). Gumroad for quick start / testing.

Avoid Etsy (not for code), Teachable (expensive, poor dev tooling).

### Automatable by Code System
Highest automation potential:
- AI prompt packs (generate systematically)
- eBooks (auto-generate with AI)
- Template bundles (package systematically)
- Spreadsheets (Python-generated)
- n8n/Make workflow templates (template + AI integration)

### Python on Gumroad — Revenue Examples
- $2,539/month (solo dev, 20X growth in 12 months, $12–$29 products + $49–$79 bundles)
- $1,500/month passive (automation scripts: PDF converters, budget calculators, Excel automation)
- $1,019 in 90 days (coding + N8N automations + AI workflow knowledge)
- $10,000+ cumulative (from consistent posting, not viral)

### Big Gaps / Unserved Opportunities
1. n8n/Make AI workflow templates (few creators can build)
2. Developer resource bundles (Chrome extensions, API kits)
3. Industry-specific spreadsheets ($29–$99)
4. Compliance/HR tools for SMBs (subscription)
5. Hyper-niche eBooks (10x price vs generic)

### Automation Stack
- Gumroad (sales/delivery/affiliates) → n8n (workflows) → Kit (email) → AI chatbot (support)
- For passive: Gumroad Workflows for email sequences, native license key delivery, automated affiliates
